# **App Name**: MyRoxas

## Core Features:

- Issue Reporting: Allow citizens to report city issues with photo upload and GPS integration.
- Appointment Booking: Enable online booking of appointments for city services with calendar integration.
- City Support Chat: Provide a chat interface for citizens to communicate with city support.
- Emergency Hotlines: Offer quick access to emergency hotlines with a prominent, glowing button.
- Service Recommendation: Suggest relevant city services based on user input using AI as a tool.

## Style Guidelines:

- Primary color: Roxas Yellow (#FFD200) for energy and civic optimism.
- Background color: Light yellow (#F7F5EB), desaturated to 20% for a subtle backdrop.
- Accent color: Cool Blue (#0077B6) for a hint of trust and technology; contrast with primary.
- Text color: Charcoal Gray (#1E1E1E) for modern text contrast.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary and precise feel.
- Body font: 'Inter', a grotesque-style sans-serif with a modern and neutral look.
- Use Lottie animations for icons to add movement without clutter (camera flash, chat bubble, map pin).
- Full-screen hero section with scroll-down reveal of features and a lively, card-based grid. ShadCN components will be used for layout.
- Subtle parallax scrolling, card float animations, section fade-ins, and an animating background for a dynamic feel. Capiz seal subtly orbits in hero.
